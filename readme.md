MCP RAG Agent
一个基于MCP（Model Context Protocol）协议和RAG（Retrieval-Augmented Generation）技术的智能代理系统，通过语义检索动态选择和执行工具。

🌟 特性
MCP协议支持: 标准化工具调用接口

RAG检索: 基于语义的工具智能选择

多服务器管理: 支持多个MCP服务器并行运行

向量化存储: 使用Milvus向量数据库存储工具信息

异步处理: 全异步架构，高性能处理

可扩展性: 轻松添加新的工具和服务器

🏗️ 系统架构

用户查询 → RAG语义检索 → 筛选相关工具 → ReAct智能体 → 工具执行 → 结果返回

组件说明
MCP服务器: 提供标准化工具接口（数学、天气等）

向量数据库: 存储工具语义信息，支持相似度检索

工具管理器: 基于RAG技术筛选相关工具

智能代理: 使用ReAct模式进行推理和工具调用
